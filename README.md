## 🚀 Installation

Clone repo

```sh
git clone https://github.com/isdi-coders-2023/Abel-Naharro-Final-Project-front-202309-mad
cd Abel-Naharro-Final-Project-front-202309-mad
```

Install dependencies

```sh
npm i
```

Run project

```sh
npm run dev
```

Run tests with coverage report

```sh
npm run test:pro
```

Ready to enjoy! 😉
